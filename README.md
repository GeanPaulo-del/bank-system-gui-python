# 💰 Sistema Bancário com Interface Gráfica em Python

Aplicação de banco desenvolvida em Python com interface gráfica utilizando Tkinter e armazenamento de dados em JSON.

## 🚀 Funcionalidades

- Login de usuário
- Visualização de saldo em tempo real
- Depósito de valores
- Saque com verificação de saldo
- Validação de entrada (evita erros)
- Salvamento automático dos dados (persistência)

## 🧠 Tecnologias

- Python
- Tkinter (interface gráfica)
- JSON (armazenamento de dados)

## ▶️ Como executar

```bash
python main.py
📁 Estrutura
main.py → aplicação principal
usuarios.json → banco de dados dos usuários
🎯 Objetivo
Projeto criado para praticar lógica de programação, desenvolvimento de interfaces gráficas e manipulação de dados persistentes em Python.
